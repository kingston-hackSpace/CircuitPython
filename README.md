# CircuitPython

Visit [circuitPython.org](https://circuitpython.org/)

What is CircuitPython? Read more [here](https://learn.adafruit.com/welcome-to-circuitpython/what-is-circuitpython))

----
### Why CircuitPython? 

CircuitPython is a version of the Python programming language designed to run on microcontroller boards.

----
### Compatibility

Run CircuitPython scripts for :

- Microcontrollers from Adafruit (Feather and Metro boards)

- ESP32 boards

- RaspberryPi Pico

- Teensy boards

- NOT suitable with Arduino Boards (Arduino runs in C++ programming language)
